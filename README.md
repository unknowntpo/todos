## Quick Start
Clone the repository
```
$  git clone https://github.com/unknowntpo/todos.git
```

Use the default `.envrc` file
```
cp .envrc.example .envrc
```

Run the project
```
$ make run/compose/up
```

Stop the project
```
$ make run/compose/down
```
